# medical-rag-chatbot
# How to run this project
### STEPS:

1. Clone this repository
```bash
 git clone https://github.com/AzanPy/medical-rag-chatbot.git
```
2. Create a virtual environment
```bash
conda create -n medical-rag-chatbot python=3.10 -y
```
3. Activate the virtual environment
```bash
conda activate medical-rag-chatbot
```
4. Install the required packages
```bash
pip install -r requirements.txt
```
